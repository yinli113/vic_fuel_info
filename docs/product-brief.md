# Product Brief

## Target Audience
Victorian drivers, commuters, and budget-conscious households looking to save money on fuel and avoid empty petrol stations.

## Primary Use Cases
1. **Find Cheapest Fuel Nearby**: Map-based and list-based search.
2. **Commuter Route Check**: Find the best fuel stop on the way to work/home without significant detours.
3. **Report/Check Outages**: Find out if a station is dry before driving there; report an outage to help the community.
4. **Price Timing**: Check historical trends to decide whether to buy today or wait for a price drop.

## Differentiation
- **Transparent Data Freshness**: Clear indication if data is 24-hours delayed (official) or real-time (community).
- **Data Trust**: Crowdsourced data is validated against official data and community consensus.
- **Hyper-Local Hourly Trends**: Rolling 1-hour window for suburb-level price and outage tracking.
